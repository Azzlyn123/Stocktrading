Task 0: Define execution timing rules (one-time “contract”)

Goal: eliminate accidental peeking forever.
	•	Intrabar exits (STOP / TARGET):
Triggered by OHLC crossing within the bar → fill uses gap-aware rules on the same bar (open/level).
	•	Close-based decisions (TIME STOP / HARD EXIT / RULE-BASED “market” exits):
Decision uses bar close → fill occurs next bar open (or next available open), never same bar close.
	•	EOD close:
If you force-flat at end of session: fill at session close only if you truly have that print available; otherwise treat as next open. (Most sims accept session close here—just document it.)
